Djangopoly Documentation

What is Djangopoly

Djangopoly is a web-based implementation of the old time classic board game Monopoly.

Technologies used:

  • HTML
  • CSS
  • JavaScript / jQuery / various external libraries
  • Python / Django

Who is this for:

People of ages between 18-99.

Types of games:

  • Public - can be played with anyone worldwide with no limitations
  • Private - only people with the link can join the game

Gameplay:

Beginning

As soon as a game is created by one of the players, other players can start joining in. When the first dice roll is made the game becomes locked and no new players can join.

Playing the game

The order of the players is determined by their time of joining, i.e. the first player to join will be the first to roll the dice. Each player gets a piece on the board and that piece has a colour associated to it that can be seen around the piece. That same colour is then used as an overlay over each property that the player buys. The overlay becomes more transparent when a property is mortgaged. As the player moves around the board they can not only land on properties, but also other special types of squares. These are: utilities, free parking, go to jail, jail, go, chance, community chest. When the player lands on chance/community chest they have the option of drawing a card that would affect the player's progress in game, e.g. they will be advanced to GO, they will pay additional tax, etc.

Ending

The game ends for a player if they can no longer pay their debts, i.e. declare bancruptcy. They can also leave the game voluntarily at any point.
